# 基于SSH的求职招聘管理系统2

#### Description
基于SSH的求职招聘管理系统拥有三个角色：用户、公司、管理员

- 用户：查看招聘信息、投递简历、留言、修改个人信息
- 公司：查看求职信息、发布招聘信息、求职资讯管理、公司信息管理
- 管理员：用户管理、公司管理、留言管理、公告管理、招聘管理、友情链接管理等
